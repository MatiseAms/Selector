# jQuery.Selector

### Initialize Selector

```javascript
$('#ID-TO-ELEMENT').Selector({
	clickToOpen: false,
	setActiveOnClick: true,
	checkedClass: 'checked',
	setOrder: true,
	transitionTime: 300,
	valueAttr: 'data-value',
	arrows: false
});
```

### Initialize Selector

```javascript
bower install selector --save-dev
```
